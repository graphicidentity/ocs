ESDC Self-Paced Document Accessibility Course

This course is based on the principles of document accessibility, drawing from the Web Content Accessibility Guidelines 2.1 (WCAG 2.1) level AA as they apply to digital documents. The concepts, success criteria, and techniques are thoroughly explained using clear and simple language, with relevant examples and sample markup. There are * modules in the course, and each module takes about * hour(s) of reading. This course is designed for document creators, designers, developers, testers, trainers, content authors, project managers, and anyone interested in making digital documents more accessible to all users, including those with disabilities..

Module 1 - Accessibility Fundamentals

Introduces the core principles of accessibility, why it matters, and the impact on users with disabilities. Covers inclusive design concepts and the legal and policy frameworks that guide accessibility.

Module 2 - Global M365 Document Techniques

Focuses on creating accessible documents in Microsoft 365 applications. Explains how to use built-in accessibility tools, apply proper structure, and ensure consistent formatting across Word, Excel, and PowerPoint.

Module 3 - PowerPoint Accessibility Techniques

Teaches how to make presentations accessible by using slide layouts, adding alternative text for images, ensuring readable fonts and color contrast, and providing meaningful slide titles.

Module 4 - Word Accessibility Techniques

Covers best practices for creating accessible Word documents, including the use of headings, styles, tables, and descriptive hyperlinks to improve navigation and readability

Module 5 - Outlook Email Accessibility Techniques

Explains how to compose accessible emails in Outlook by structuring content clearly, using accessible fonts, and adding alt text for images. Emphasizes clarity in subject lines and message hierarchy.

Module 6 - Introduction to Accessible PDFs

Provides an overview of what makes a PDF accessible, including tagging, reading order, and the use of accessibility checkers to verify compliance.

Module 7 - Exporting Word to PDF

Shows how to export Word documents to PDF while preserving accessibility features such as headings, alt text, and document structure.

Module 8 - Exporting PowerPoint to PDF

Demonstrates how to export PowerPoint presentations to accessible PDFs, ensuring that visual and textual content remains usable for all audiences.

Module 9 - Foxit PDF Editor for Accessibility

Explains how to use Foxit PDF Editor to enhance the accessibility of existing PDFs, including adding tags, fixing reading order, and validating compliance.
