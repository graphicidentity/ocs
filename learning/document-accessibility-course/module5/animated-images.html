Designing with animated images

An animated Image (including video) can distract users, particularly those with Attention Deficit Disorder. If it includes flashes, the animation can also cause seizures among users with photosensitive epilepsy.

Follow these Best practices for animated images:

  • Ensure the animation stops after five seconds or provide users with a pause button.
  • Ensure the animation doesn’t flash more than three times per second.

For more information on flashing content, see Module 9: Audio, video, animations and motion > Flashing content.
