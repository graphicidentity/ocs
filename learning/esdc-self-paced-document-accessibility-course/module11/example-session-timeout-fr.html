En raison de l'inactivité, votre session expirera dans 2 minutes. Si vous avez besoin de plus de temps, veuillez cliquer sur "Continuer la session". Pour permettre à cette session de se terminer, cliquez sur "Terminer la session".

Dialogue accessible sur le délai d'attente de la session

Après 15 secondes d’inactivité, JavaScript est activé et déplace le focus de l’utilisateur vers une boîte de dialogue contextuelle qui utilise l’attribut role="alertdialog". L’utilisateur peut choisir de continuer ou de terminer la session. Si l’utilisateur clique sur Continuer la session, JavaScript réinitialise le compteur de 15 secondes. Si l’utilisateur clique sur Terminer la session, JavaScript quitte la page.
